## Requirements

- Java 11 or higher  
- JavaFX SDK installed and linked in IntelliJ IDEA  
- IntelliJ IDEA or any Java IDE  

---

## How to Run

1. Open the project in IntelliJ IDEA  
2. Make sure JavaFX SDK is added to project libraries  
3. Run AeroWeather.java  
4. The app window will open with GUI and CSS applied
